## 1. Windows設定

### CorvusSKK

1. https://github.com/nathancorvussolis/corvusskk/releases/latest から `corvusskk-X.Y.Z.exe` をダウンロードして実行
2. インストール後再起動が必要
3. 設定 → 時刻と言語 → 言語と地域 → 日本語の「言語のオプション」→ キーボードの追加 → **CorvusSKK** を追加
4. 設定 → 時刻と言語 → 言語と地域 → 入力 → キーボードの詳細設定 → 規定の入力方式の上書き → 日本語 - CorvusSKK
5. 設定については下記を参考に
   https://zenn.dev/toriwasa/articles/327d11c45a62e8
   ただし、動作1の「初期入力モード」についてはASCIIとしておく
   WSL2で作業する際にIMEの切り替え無しでCtrl+jを有効にするため

### SandSExe

1. https://github.com/fiore57/SandSExe/releasesl からダウンロード・解凍し任意の場所に配置（例: `%USERPROFILE%\.config\skk\SandSExe\`）
2. `Win+R` → `taskschd.msc` → タスクスケジューラを起動
3. 右ペインの「タスクの作成」を選択
4. 各タブを以下のように設定：

| タブ | 項目 | 設定値 |
|---|---|---|
| 全般 | 名前 | `SandSExe` |
| 全般 | 最上位の特権で実行する | チェックを入れる |
| 全般 | 構成 | `Windows 10` |
| トリガー | タスクの開始 | `ログオン時` |
| トリガー | 対象ユーザー | 自分のユーザー |
| 操作 | 操作 | `プログラムの開始` |
| 操作 | プログラム/スクリプト | `SandSExe.exe` のフルパス |
| 操作 | 開始(オプション) | SandSExe.exe があるディレクトリ |
| 条件 | 全項目 | チェックを外す |
| 設定 | タスクを停止するまでの時間 | チェックを外す |

5. OKで登録

:::message
PC起動時に `runtime_error: failed to update notifyicon` のダイアログが表示される場合は、トリガーに遅延を設定する必要がある
:::

### crvskkserv

1. https://github.com/nathancorvussolis/crvskkserv/releases/latest から `crvskkserv-X.Y.Z-x64.zip` をダウンロード
2. 任意の場所に解凍（例: `%USERPROFILE%\.config\skk\crvskkserv\`）
3. `Win+R` → `shell:startup` → 開いたフォルダに `crvskkserv.exe` のショートカットを置く
4. タスクトレイのアイコンから設定を開く：
    - **ポート:** `1178`（デフォルト）
    - 「ループバックのみ」にチェック
    - **GoogleCGIAPI追加** ボタンを押してOK
 5. CorvusSKKの「辞書1」でSKK辞書サーバーの「使用する」をON

## ~~2. WSL2設定~~

~~デフォルトのNATモードではWSL2からWindowsのlocalhostに直接繋がらないため、ミラーモードに変更する。~~

```powershell
# PowerShell（管理者権限）
@"
[wsl2]
networkingMode=mirrored
"@ | Out-File $env:USERPROFILE\.wslconfig -Encoding utf8

wsl --shutdown
```
